## 🎨 Design-Details

- **Farben**: 
  - Dunkelblau (#1a1a2e) - Header & Buttons
  - Rot (#e94560) - Preise & Hover-Effekte
  - Weiß & Hellgrau - Hintergrund & Cards
- **Schriftart**: Segoe UI (moderne, gut lesbare Schriftart)
- **Layout**: Responsive Grid (passt sich an Bildschirmgröße an)

## ✏️ Website anpassen

Um Produkte zu ändern, editieren Sie einfach die `index.html`:

1. Gehen Sie zum Repository auf GitHub
2. Klicken Sie auf `index.html`
3. Klicken Sie auf das Bleistift-Icon (Edit)
4. Passen Sie Produktnamen, Preise und Beschreibungen an
5. Speichern Sie die Änderungen (Commit)

Die Änderungen sind sofort auf Ihrer Website sichtbar!
